Icons

The icons used are one-color and two-dimensional. They are provided as a web font which is to be found in the resources as a TrueType font.

For * icons.scss

For inside-icon element, use the global .icon and .icon--before or .icon--after class followed by the more specific class.

My document
My document 
<!-- only-icon element -->
<span class="icon icon--pdf"></span> My document <br>

<!-- inside-icon element -->
<a href="doc.pdf" class="icon icon--before icon--pdf">My document</a>
